# 🌱 SIAD EIRE

**Sistema de Informações e Análise de Dados para Apoio à Tomada de Decisão em Propriedades Rurais**

O SIAD (Sistema de Informações e Análise de Dados) é uma aplicação web desenvolvida no contexto do grupo de pesquisa EIRE, com o objetivo de integrar, visualizar e analisar dados georreferenciados e atributos técnicos de propriedades rurais.

A ferramenta apoia a tomada de decisão relacionada ao uso de recursos energéticos, hídricos e produtivos, promovendo uma visão integrada e contextualizada das propriedades analisadas.

---

## 🎯 Objetivo

Disponibilizar uma plataforma interativa que permita:

- Visualização de dados geoespaciais em mapas  
- Análise integrada (produtiva, ambiental e energética)  
- Geração de indicadores e relatórios  
- Apoio à tomada de decisão baseada em dados  

---

## ⚙️ Funcionalidades

- 📍 Visualização de propriedades rurais em mapa interativo  
- 🗺️ Exibição de dados georreferenciados (pontos e polígonos)  
- 🧩 Visualização por camadas temáticas (energia, cultivo, irrigação etc.)  
- 📊 Dashboard com indicadores analíticos  
- 🧠 Integração com módulo de Sistema de Apoio à Decisão (SAD)  
- 📄 Geração de relatórios em PDF  
- 📌 Visualização consolidada de dados por propriedade  

---

## ⚠️ Importante

O sistema não permite inserção de dados pela interface.  
Os dados são inseridos manualmente pela equipe do projeto, sendo o SIAD responsável apenas pela leitura, análise e visualização.

---

## 🏗️ Arquitetura

O SIAD segue uma arquitetura web modular baseada no modelo C4, composta por:

- Front-end: HTML, CSS, JavaScript  
- Back-end: PHP  
- Banco de Dados: MySQL (XAMPP)  
- Serviços externos: API do Google Maps  

A arquitetura foi projetada de forma modular e desacoplada, permitindo:

- Evolução contínua do sistema  
- Integração futura com o SIPR-Geo  
- Inclusão de novos módulos analíticos  

---

## 🗄️ Dados

- Banco de dados relacional inspirado no modelo do ArcGIS Pro  
- Dados atuais são fictícios (uso para validação do protótipo)  
- Alimentação manual realizada pela equipe do projeto
